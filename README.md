Este proyecto consiste en el desarrollo de una aplicación web que permite gestionar la reserva de instalaciones deportivas de un centro: pistas de pádel, tenis, baloncesto y fútbol sala.
El objetivo es ofrecer una plataforma sencilla y accesible para que los usuarios puedan consultar la disponibilidad de las pistas y realizar reservas de manera rápida y segura.

El proyecto forma parte del trabajo intermodular del ciclo DAM e integra contenidos de diferentes módulos: programación, bases de datos, entornos de desarrollo, lenguaje de marcas y sistemas.
